# Bandit Level 4 – Commands Used

## ls
- Lists files in current directory

```bash
ls
```

## ls -la
- Lists all files including hidden ones with details

```bash
ls -la
```

## cat
- Displays file content

```bash
cat ./-file00
cat ./-file01
cat ./-file02
cat ./-file03
cat ./-file04
cat ./-file05
cat ./-file06
cat ./-file07
cat ./-file08
cat ./-file09
```
## lsfile
- Shows the type of each file

```bash
file ./*
```

## grep
- Filters output to find readable file

```bash
file ./* | grep ASCII
```
