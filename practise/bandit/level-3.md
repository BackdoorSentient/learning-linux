# Bandit Level 3 – Commands Used

## ls
- Lists files in current directory

```bash
ls
```

## cd
```bash
cd inhere
```
## ls -la
- Lists all files including hidden ones with details
```bash
ls -la
```

## cat
-Displays file content

```bash
cat ...Hiding-From-You
```
## pwd
-Shows current directory path

```bash
pwd
```
